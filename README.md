# PUMI-PAGE — Conectado a PUMI-API

Esta versión usa el backend privado:

https://pumi-api.onrender.com

## Subir

Descomprima el ZIP y suba todo el contenido directamente a la raíz de `PUMI-PAGE`, reemplazando los archivos existentes.

Después espere el despliegue de GitHub Pages y actualice con `Ctrl + F5`.

## Incluido

- Login real con `PUMI_USUARIOS`.
- Sesión JWT.
- Lectura real de actividades.
- Lectura real del resumen.
- Lectura real de catálogos y delegaciones.
- Dashboard y mapa con datos reales.
- Menú según rol.
- Cierre de sesión.

El usuario debe existir en `PUMI_USUARIOS` con los campos configurados en Render.
